🎓 Expert Booking App
Expert Booking App is a consultant-matching platform designed to connect users with professionals across various industries. This project was my first entry into the world of mobile development, focusing on UI construction and efficient navigation.

📖 Project Overview
The app solves the problem of finding specialized advice by categorizing experts into fields. Users can browse categories, view detailed expert profiles, and schedule consultation meetings.

✨ Key Features
Field Categorization: Browse experts based on their specific industry or field of study.

Expert Discovery: View a curated list of professionals with their credentials and ratings.

Booking System: Select a professional and book a dedicated time slot for a meeting.

Search & Filter: Quickly find specific experts within a selected field.

🏗 Technical Focus
As my introductory project, the focus was on mastering the core pillars of Flutter:

Vanilla Flutter: Built using standard Dart and built-in Widgets without external state management libraries.

Navigation: Implementation of named routes and constructor-based data passing between screens.

UI/UX: Practicing layout constraints using Rows, Columns, ListViews, and custom-styled Containers.

Data Modeling: Creating clean Dart classes to represent "Experts" and "Fields."

📂 Project Structure
Plaintext
lib/
